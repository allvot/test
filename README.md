## Installation

### Prerequisites

- Docker
- Docker Compose

### Instructions

Run the foollo

```bash
# startup the servers:
docker-compose up -d

# spy the app's logs
docker-compose logs -f

# stop the services
docker-compose down -v
```

## Usage

- Run docker environment
- Open your browser
- Visit http://localhost:3001
- Use the app
